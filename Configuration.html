Configuration

In this part of the new Wiki page, we are going to list a few different config.toml files, already setup for different usecases. For example a file with the netpos and webgps map enabled by default and so on, you get the idea.

However its quite time consuming to create such files in larger quantities, so if you want to add additional tomls, create a pull request on Github. Then it will eventually be shown on the website. (If its working, not a duplicate and accepted)

Learn more about the original default.toml on EvilSockets Github page..
